# Book Store PHP

Welcome to the **Book Store PHP** project! This is a web application that allows users to browse, purchase, and manage their book collections. Built using PHP, CSS, and JavaScript, this application provides a seamless shopping experience.

## Features

- User registration and authentication
- Browse books by category
- Add books to the shopping cart
- Checkout process
- Admin panel for managing products and orders
- Responsive design for mobile and desktop devices

## Technologies Used

- **PHP**: Server-side scripting language for backend development
- **CSS**: Styling the application for a user-friendly interface
- **JavaScript**: Enhancing user interaction and experience
- **MySQL**: Database management for storing user and product information

Usage
User: Register an account to start browsing books, adding them to your cart, and making purchases.
Admin: Log in to the admin panel to manage products and view orders.
